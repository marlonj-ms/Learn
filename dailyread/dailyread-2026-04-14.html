链表与 Add Two Numbers

2026-04-14 · 阅读时间约 4 分钟

1. 一句话总结

单向链表是一种"自引用"的数据结构：每个节点除了存数据，还存"下一个节点的引用"。

2. ListNode 节点定义

public class ListNode
{
    public int val;            // 当前节点存的值
    public ListNode next;      // 指向下一个节点（同类型！）

    public ListNode(int val = 0, ListNode next = null)
    {
        this.val = val;
        this.next = next;
    }
}

关键点：next 字段的类型是 ListNode 本身 —— 这就是"自引用类（self-referential class）"。

3. 取模 % 与进位

int a = 17;
int sum = a;
int carry = sum / 10;    // 进位：17 / 10 = 1
int digit = sum % 10;    // 本位：17 % 10 = 7
整数溢出风险：用 int 拼接大数（如把整个链表转成 int 相加再拆开），如果数字超过 int.MaxValue 会溢出。 正确做法是逐位相加 + 维护进位。

4. Add Two Numbers 核心逻辑

ListNode AddTwoNumbers(ListNode l1, ListNode l2)
{
    ListNode dummy = new ListNode(0);
    ListNode current = dummy;
    int carry = 0;

    while (l1 != null || l2 != null || carry > 0)
    {
        int sum = carry
                + (l1 != null ? l1.val : 0)    // ← 三元运算符 + null 检查
                + (l2 != null ? l2.val : 0);

        carry = sum / 10;
        current.next = new ListNode(sum % 10);
        current = current.next;

        l1 = l1?.next;     // ← null 条件访问
        l2 = l2?.next;
    }

    return dummy.next;
}

5. 三元运算符（ternary operator）

条件 ? 真时的值 : 假时的值

int abs = n < 0 ? -n : n;
int safeVal = node != null ? node.val : 0;

6. 自测

问题答案
17 % 10 = ?7
17 / 10 = ?（int 除法）1
什么是 dummy node？占位用的头节点，避免在循环里特判"是不是第一个节点"
l1?.next 当 l1 是 null 时返回什么？null（不会崩）
为什么循环条件要加 || carry > 0？两个链表都遍历完后，可能还有最后一位进位（如 99+1）
